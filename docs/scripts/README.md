# CLI Scripts Index

This directory documents the entry-point CLI runners and directs to specialized documentation.

* **`run_pipeline.py`**: The master orchestrator. Runs `run_profiling.py` followed by `run_reporting.py` to execute a full pipeline run.
* **`run_profiling.py`**: Entry-point for profiling raw data. (See [Profiling Scripts](file:///home/vicbguti/Projects/vehicles/docs/scripts/profiling.md) for details).
* **`run_reporting.py`**: Entry-point for generating analytical reports. (See [Reporting Scripts](file:///home/vicbguti/Projects/vehicles/docs/scripts/reporting.md) for details).
