# Volume Report Generators

This directory contains Python scripts that parse the database caches and generate the markdown reports under `reports/02_volume/`.

## 📂 Scripts

* [storage.py](./storage.py) — Builds the storage.md report.
* [growth_trends.py](./growth_trends.py) — Builds the growth_trends.md report.
* [memory_profile.py](./memory_profile.py) — Builds the memory_profile.md report.
