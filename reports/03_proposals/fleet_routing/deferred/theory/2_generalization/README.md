# Formal Algebraic Generalizations

This directory contains formal mathematical formulations for routing and packing scenarios, generalizing the problem to arbitrary vehicle pool sizes, capacities, and constraints.

## 📂 Sub-Directories

### 1. [Search Space](./1_search_space)
Combinatorial search space permutations, multi-class candidate selection formulas, and cancellation proofs.

### 2. [Constraints](./2_constraints)
Algebraic proofs of vehicle class boundaries, pool availability intersections, range vs. point collapse conditions, and subset pruning behavior.

### 3. [Partitioning](./3_partitioning)
Generalized binary decision matrix formulation for mapping selected vehicles onto carrier fleets (bin packing constraints).
